1) ES6 класи мають простіший синтаксис, вбудовані extends та super, а також автоматично працюють у strict mode.
2) Прототипне наслідування базується на посиланнях між об'єктами, де дочірній об'єкт шукає методи в ланцюжку прототипів батьків.
3) У методах об'єктів this вказує на сам об'єкт, тоді як у звичайних функціях він залежить від способу виклику або дорівнює undefined у strict mode.
4) Strict mode забороняє використання this як глобального об'єкта, що вимагає обов'язкового виклику конструкторів через new.
5) Геттери та сеттери дозволяють контролювати доступ до властивостей, додавати валідацію та приховувати внутрішню логіку даних.
6) Для збереження контексту слід використовувати метод bind(), стрілкові функції або обгортки.
7) Наслідування та поліморфізм підтримуються повноцінно, інкапсуляція - переважно через угоди про іменування, а абстракція на рівні мови відсутня.
